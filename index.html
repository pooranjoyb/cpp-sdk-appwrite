Appwrite C++ SDK Documentation

Introduction

The Appwrite C++ SDK is a client library for communicating with the Appwrite backend. It provides a clean, well-documented interface for working with:

  • User accounts
  • Databases and documents
  • Storage buckets and files
  • Messaging topics and subscriptions
  • Health checks and system diagnostics

Key Features

  • Easy-to-use API abstraction
  • Cross-platform compatibility
  • Fully documented with Doxygen
  • Modular service separation (Account, Database, Storage, etc.)
